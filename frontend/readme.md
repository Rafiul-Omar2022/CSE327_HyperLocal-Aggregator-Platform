# Frontend

- Install fnm (Node js version manager)
    - doc: `https://github.com/Schniz/fnm`
- Before install node module select node version. We are using node `v24.18.0` LTS
- Ppen powershell for the code command execution and type: `fnm env --use-on-cd --shell powershell | Out-String | Invoke-Expression`
- Then type `fnm use` for using node js 24.18.0 version
- For installing node dependencies type: `npm install`


## How to run frontend:
- Type: `npm run dev`

### dependencies:
- React
- Typescript
- Tailwindcss
- scaffolding: vite
